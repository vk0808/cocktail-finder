Cocktail finder is a React-based application which uses Cocktail DB API to fetch cocktail.

## Dependencies

- react 17.0.1
- react-dom 17.0.1
- react-loader-spinner 4.0.0
- react-router-dom 5.2.0
- react-scripts 4.0.0

## Quick start:

### Install Dependencies

```sh
$npm install
```

### Run development server

```sh
$npm run dev-server
```

### Build the project

```sh
$npm run build
```
